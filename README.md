# Credit Risk Analysis and Loan Default Prediction Using Machine Learning
Proyek ini bertujuan untuk menganalisis risiko kredit dan memprediksi kemungkinan gagal bayar pinjaman (loan default prediction) menggunakan pendekatan machine learning. Analisis dilakukan untuk membantu institusi keuangan dalam meningkatkan kualitas pengambilan keputusan kredit, mengurangi risiko kredit macet (Non-Performing Loan/NPL), serta mendukung strategi manajemen risiko berbasis data. Dalam proyek ini dilakukan tahapan lengkap data science workflow mulai dari data preprocessing, exploratory data analysis (EDA), feature engineering, feature selection, pelatihan model machine learning, hingga evaluasi performa model.

## Objectives
- Menganalisis karakteristik dan perilaku nasabah terkait risiko kredit
- Memprediksi kemungkinan gagal bayar pinjaman menggunakan machine learning
- Membandingkan performa beberapa algoritma machine learning
- Memberikan insight dan rekomendasi bisnis untuk mitigasi risiko kredit
- Mendukung proses pengambilan keputusan kredit yang lebih efektif dan efisien

## Project Workflow
### Data Preprocessing
- Menangani missing values
- Menghapus data duplikat dan outlier
- Encoding data kategorikal
- Feature scaling dan transformasi data

### Exploratory Data Analysis (EDA)
- Analisis distribusi data nasabah
- Visualisasi hubungan antar variabel
- Analisis pola gagal bayar pinjaman
- Identifikasi faktor yang mempengaruhi risiko kredit

### Feature Engineering & Feature Selection
- Pemilihan fitur yang paling relevan
- Mengurangi fitur yang kurang berpengaruh
- Meningkatkan kualitas input model machine learning

### Machine Learning Modeling
Beberapa algoritma machine learning digunakan dan dibandingkan performanya, seperti:
- Logistic Regression
- Random Forest
- XGBoost dengan pengolahan imbalance data menggunakan SMOTE

### Model Evaluation
Evaluasi model dilakukan menggunakan beberapa metrik:
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score

## Business Insights
Hasil analisis menunjukkan bahwa machine learning dapat membantu institusi keuangan dalam:
- Mengidentifikasi nasabah berisiko tinggi lebih awal
- Mengurangi potensi kredit macet
- Meningkatkan efektivitas proses approval kredit
- Mendukung strategi mitigasi risiko berbasis data

## Conclusion
Proyek ini menunjukkan bahwa pendekatan machine learning dapat digunakan secara efektif untuk analisis risiko kredit dan prediksi gagal bayar pinjaman. Dengan memanfaatkan data historis dan teknik analisis yang tepat, institusi keuangan dapat meningkatkan kualitas pengelolaan risiko dan mendukung pengambilan keputusan yang lebih akurat.
