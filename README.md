# Credit-Risk-Analysis-and-Loan-Default-Prediction-Using-Machine-Learning
Proyek ini bertujuan menganalisis risiko kredit dan memprediksi gagal bayar pinjaman menggunakan machine learning. Tahapan meliputi preprocessing data, EDA, feature selection, pelatihan model, dan evaluasi performa untuk membantu institusi keuangan meningkatkan keputusan kredit dan mitigasi risiko.
